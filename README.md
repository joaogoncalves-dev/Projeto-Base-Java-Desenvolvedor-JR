# Projeto Base Java - Desenvolvedor JR

Diário de bordo de 90 dias de estudos intensivos em Java, focado em construir uma boa base.

##  OBJETIVO E METODOLOGIA DE ESTUDO

Este repositório é meu **Diário de Estudos e Prova de Conceito**. O objetivo é demonstrar minha disciplina diária e meu conhecimento nos fundamentos da linguagem Java.

**Por que o código está cheio de comentários?**

Cada arquivo (Dia 1, Dia 2, etc.) é intencionalmente documentado para servir como **material de revisão**.

* **Exemplos de Documentação:**
    1.  **Sintaxe:** Explicação detalhada do uso de operadores (`==`, `&&`) e palavras-chave.
    2.  **Boas Práticas:** Notas sobre convenções (`camelCase`) e o uso de `final` para constantes.
    3.  **Lógica:** O passo a passo de como estruturas complexas (`if/else` e `while`) foram construídas
